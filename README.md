# Tap Tower 🗼

One-tap аркада для Telegram Mini App. Строй башню, ставь блоки точнее → лезь выше.
Монетизация — Telegram Stars (косметика + «продолжить»). Жанр Stack/Ketchapp — проверенно аддиктивный + вирусный в Telegram.

**Зачем:** самый низкий платёжный барьер для несовершеннолетнего автора — Telegram сам
принимает деньги (Stars), без ИП/Stripe. Дистрибуция вирусная (шеринг результата в чат).

## Статус

- ✅ **MVP-движок** (`index.html`) — полностью играбельный: canvas-рендер, slice-механика,
  perfect-комбо, счёт, рекорд (localStorage), game-over, рестарт, Telegram WebApp SDK
  (ready/expand/HapticFeedback/тема), кнопки шеринга и Stars (заглушки).
- ⬜ Бэкенд лидерборда (FastAPI + initData HMAC — переиспользуем паттерн из Doday)
- ⬜ Telegram Stars: `tg.openInvoice` на 1 Star для «продолжить» + покупка скинов
- ⬜ Скины/темы башни (косметика за Stars)
- ⬜ Лидерборд друзей + «побей мой результат» с deep-link
- ⬜ Деплой (статика + бэкенд на тот же VPS, или Cloudflare Pages для статики)

## Локальный запуск

```bash
cd tap-tower
python -m http.server 8090
# открой http://127.0.0.1:8090/
```

## Монетизация (Telegram Stars)

| Товар | Цена | Механика |
|---|---|---|
| Продолжить после проигрыша | 1 ⭐ | `tg.openInvoice` → продолжить с текущей высоты |
| Скин/тема башни | 25-75 ⭐ | косметика, разблокируется навсегда |
| (позже) Премиум — без рекламы | 150 ⭐ | если добавим rewarded-ads |

Вывод Stars → у Telegram свои пороги/правила; на этапе приёма физлицу можно.

## Что нужно от тебя (один раз)

1. В @BotFather создать бота (имя/username для игры), получить токен.
2. Включить для бота приём Telegram Stars (BotFather → Payments, без сторонних провайдеров для Stars).
3. Привязать Mini App к боту (кнопка меню → URL игры).
4. Прислать мне токен бота — остальное (бэкенд, Stars-инвойсы, лидерборд, деплой) делаю я.

## Стек

Vanilla JS + Canvas (фронт игры, без сборки) · Telegram WebApp SDK ·
бэкенд (план) — FastAPI + asyncpg, как в Doday.
